# INFO6105 Course Bot

AI Teaching Assistant for **Data Science Engineering Methods** (INFO6105), Northeastern University.

A chatbot grounded in your actual course materials — syllabus, lecture transcripts, and an auto-generated concept map. It answers the way a well-prepared TA would, using Professor Claster's own explanations and examples.

## Quick Start

### 1. Install dependencies

```bash
cd course-bot
pip install -r requirements.txt
```

### 2. Set your Anthropic API key

```bash
# macOS / Linux
export ANTHROPIC_API_KEY="sk-ant-..."

# Windows (PowerShell)
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

You can get a key at [console.anthropic.com](https://console.anthropic.com/).

### 3. Add your course materials

- Paste your syllabus into `knowledge/syllabus.md`
- Drop lecture transcript `.txt` files into `knowledge/transcripts/` (name them `lecture_01.txt`, `lecture_02.txt`, etc.)

### 4. Generate the concept map

```bash
python tools/build_map.py
```

This reads your syllabus and transcripts, sends them to Claude, and produces `knowledge/concept_map.json`. Review the JSON and edit it if needed — it's yours to curate.

### 5. Run locally

```bash
uvicorn main:app --reload --port 8000
```

Open [http://localhost:8000](http://localhost:8000) in your browser.

## Project Structure

```
course-bot/
├── main.py                  # FastAPI app — chat endpoint + serves frontend
├── knowledge/
│   ├── __init__.py          # Knowledge base loader (syllabus, transcripts, concept map)
│   ├── syllabus.md          # Your course syllabus (paste here)
│   ├── transcripts/         # Lecture transcript .txt files go here
│   └── concept_map.json     # Auto-generated by build_map.py
├── prompts/
│   └── system_prompt.py     # Builds the system prompt from all materials
├── tools/
│   └── build_map.py         # Script to auto-generate concept_map.json
├── static/
│   └── index.html           # Chat UI (single file, vanilla JS)
├── requirements.txt
├── vercel.json              # Vercel deployment config
└── README.md
```

## Deploy to Vercel

1. Push the project to a GitHub repo.
2. Import the repo in [vercel.com](https://vercel.com/).
3. In Vercel project settings, add the environment variable `ANTHROPIC_API_KEY`.
4. Deploy. Vercel uses `vercel.json` to route requests to the FastAPI app.

## Linking from Canvas

To make the bot accessible from your Canvas course, go to the relevant Canvas Module, click **+** → **External URL**, paste the deployed URL (e.g. `https://your-app.vercel.app`), give it a title like "AI Teaching Assistant", and check "Load in a new tab". Students can then launch the bot directly from Canvas.
